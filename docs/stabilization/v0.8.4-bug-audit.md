## Systemic Recommendations

1. **Ban `.unwrap_or_default()` on any `Result` from `ironclad_db::*`** in production — must propagate with `?`, return HTTP error, or log with `tracing::error!`
2. **Ban `.ok()` on database write operations** — silent write failure is data loss
3. **Require all `let _ =` on `Result` types to have a justifying comment** or replace with logged error
