# v0.8.9 Release Notes

Remediation release fixing 18 bugs found during exhaustive UAT and static analysis of the v0.8.8 codebase — 3 HIGH security, 6 HIGH functional, 7 MED, 2 LOW across 7 batches.

## Highlights

- **HIGH: SSE streaming fixed**: OpenAI-format tool-use deltas with `content: null` were silently dropped, breaking function-call streaming. Now emits empty-string deltas matching the Anthropic and Google arms.
- **HIGH: RwLock contention eliminated**: Config read-lock was held for the full duration of streaming LLM calls, blocking all config writes. Now clones needed values and drops the lock pre-call.
- **HIGH: CSS selector injection blocked**: Browser actions now validate CSS selectors against rule injection via `{`/`}` characters and enforce a 500-char length limit.
- **MED: WebSocket idle timeout**: Idle WebSocket clients no longer hold file descriptors and broadcast receivers indefinitely — ping/pong heartbeat with 90s timeout.
- **MED: CLI --json flag**: `ironclad status --json` and other commands now respect the `--json` flag instead of silently ignoring it.
- **MED: Live config get**: `ironclad config get` now queries the running server first, falling back to on-disk TOML only when offline.
