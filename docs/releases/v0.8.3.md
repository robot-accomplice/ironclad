# v0.8.3 Release Notes

## Overview

Test coverage hardening and package distribution release. Adds 100+ integration tests across ironclad-server API routes and ships Homebrew/Winget package manager support.

## Test Coverage Improvements

Added comprehensive integration tests for all major API route categories:

| Category | New Tests | Coverage |
|----------|-----------|----------|
| Sessions (create, list, delete, turns) | ~15 | Success + error paths, pagination |
| Turns (model selection, history) | ~10 | Per-turn selection, list events |
| Interviews (start, respond, status) | ~8 | Lifecycle, expired sessions, validation |
| Feedback (submit, list, aggregation) | ~10 | Valid/invalid ratings, dedup |
| Skills (list, toggle, catalog) | ~12 | Toggle PUT, forbidden for builtins, catalog install |
| Channels (webhooks, status, dead letters) | ~12 | Telegram/WhatsApp auth, HMAC validation, replay |
| Admin (breaker, config, diagnostics) | ~10 | Reset, validation, error formats |
| Memory & Cron | ~15 | CRUD, search, trigger, validation |
| Context & Approvals | ~8 | Context window, approval lifecycle |

**Workspace totals**: 3,316 tests passing, 0 failures, 0 clippy warnings.

## Package Distribution

- Homebrew tap for macOS/Linux (`brew install robot-accomplice/tap/ironclad`)
- Winget manifest for Windows distribution

## Version

- Workspace version bumped from 0.8.1 → 0.8.3 (0.8.2 was the stabilization bug-fix release)
- All 11 crates at v0.8.3
