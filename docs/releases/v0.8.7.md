# v0.8.7 Release Notes

Bug hunt release fixing 17 bugs across 10 crates — 1 critical, 4 high, 10 medium, 2 low severity. Also unblocks the v0.8.6 release by adding missing release notes.

## Highlights

- **CRIT: Cron jobs restored**: All cron-scheduled jobs were silently never firing due to a timestamp format mismatch between the cron worker and the scheduler's RFC 3339 parser
- **HIGH: 3 UTF-8 panics fixed**: Telegram message chunking, keystore key redaction, and platform name sanitization could all panic on multi-byte characters
- **HIGH: LLM streaming auth fix**: Streaming requests to query-auth providers (Google Generative AI) were broken — the `query:` auth mode was only implemented for non-streaming
- **HIGH: Yield engine safety**: Aave aToken balance query could panic on large values; USDC amount conversion could silently saturate
- **Release unblocked**: Created missing `docs/releases/v0.8.5.md` and `v0.8.6.md` that were preventing the release doc gate from passing
