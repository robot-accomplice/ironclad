# ironclad-wallet

Ethereum wallet with x402 payment protocol (EIP-3009), treasury policy engine, and DeFi yield optimization (Aave/Compound on Base) for the [Ironclad](https://github.com/robot-accomplice/ironclad) agent runtime.

## Usage

```toml
[dependencies]
ironclad-wallet = "0.1"
```

## Documentation

API docs are available on [docs.rs](https://docs.rs/ironclad-wallet).
