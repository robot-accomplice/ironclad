# ironclad-browser

Headless browser automation via Chrome DevTools Protocol (CDP) for the [Ironclad](https://github.com/robot-accomplice/ironclad) agent runtime. Supports navigate, click, type, screenshot, evaluate, and PDF export.

## Usage

```toml
[dependencies]
ironclad-browser = "0.1"
```

## Documentation

API docs are available on [docs.rs](https://docs.rs/ironclad-browser).
