# ironclad-core

Shared types, config parsing, personality system, and error types for the [Ironclad](https://github.com/robot-accomplice/ironclad) agent runtime.

## Usage

```toml
[dependencies]
ironclad-core = "0.1"
```

## Documentation

API docs are available on [docs.rs](https://docs.rs/ironclad-core).
