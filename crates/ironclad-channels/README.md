# ironclad-channels

Chat adapters (Telegram Bot API, WhatsApp Cloud API, Discord, WebSocket) and zero-trust Agent-to-Agent protocol (ECDH session keys, AES-256-GCM) for the [Ironclad](https://github.com/robot-accomplice/ironclad) agent runtime.

## Usage

```toml
[dependencies]
ironclad-channels = "0.1"
```

## Documentation

API docs are available on [docs.rs](https://docs.rs/ironclad-channels).
