# ironclad-schedule

Unified cron/heartbeat scheduler with DB-backed leases, mutual exclusion, and wake signaling via mpsc channels for the [Ironclad](https://github.com/robot-accomplice/ironclad) agent runtime.

## Usage

```toml
[dependencies]
ironclad-schedule = "0.1"
```

## Documentation

API docs are available on [docs.rs](https://docs.rs/ironclad-schedule).
