# ironclad-plugin-sdk

Plugin trait, manifest parser, script runner, and plugin registry with auto-discovery and hot-reload for the [Ironclad](https://github.com/robot-accomplice/ironclad) agent runtime.

## Usage

```toml
[dependencies]
ironclad-plugin-sdk = "0.1"
```

## Documentation

API docs are available on [docs.rs](https://docs.rs/ironclad-plugin-sdk).
